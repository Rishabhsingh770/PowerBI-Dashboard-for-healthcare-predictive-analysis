# 🏥 Healthcare Predictive Analytics Dashboard

## 📌 Project Overview

The **Healthcare Predictive Analytics Dashboard** is an interactive Power BI project designed to analyze patient health records and uncover patterns associated with **heart disease risk**. Using the **Heart Failure Prediction Dataset** from Kaggle, this project transforms raw healthcare data into meaningful visual insights that support data-driven clinical decision-making.

The project combines:

- ✅ Data Cleaning & Preprocessing
- ✅ Healthcare Data Analytics
- ✅ KPI Creation using DAX
- ✅ Interactive Power BI Dashboards
- ✅ Predictive Healthcare Insights

The dashboard enables healthcare professionals and analysts to identify disease trends, evaluate patient risk factors, and make informed decisions through interactive visualizations.

# 📊 Dataset Features

| Feature | Description |
|----------|-------------|
| Age | Patient age |
| Sex | Gender |
| ChestPainType | Type of chest pain |
| RestingBP | Resting blood pressure |
| Cholesterol | Serum cholesterol |
| FastingBS | Fasting blood sugar |
| RestingECG | Resting ECG results |
| MaxHR | Maximum heart rate |
| ExerciseAngina | Exercise-induced angina |
| Oldpeak | ST depression induced by exercise |
| ST_Slope | Slope of the exercise ST segment |
| HeartDisease | Target variable (0 = No Disease, 1 = Heart Disease) |

# 🔍 Key Insights

Analysis of the dataset revealed several important healthcare insights:

- 📌 Approximately **55%** of patients were diagnosed with heart disease.
- 📌 Male patients showed a higher prevalence of heart disease than female patients.
- 📌 Patients with **ASY (Asymptomatic)** chest pain had the highest disease occurrence.
- 📌 Elevated cholesterol levels were associated with increased heart disease risk.
- 📌 High resting blood pressure contributed to higher disease prevalence.
- 📌 Older patients demonstrated a greater likelihood of heart disease.
